# Changelog - Mi Proyecto AI

## [1.0.0] - 2026-03-21

### ✨ Agregado
- Estructura inicial del proyecto
- Backend en FastAPI con endpoints principales
- Frontend en React + Vite
- Integración con OpenAI GPT
- Integración con Google Gemini
- Integración con Moonshot Kimi
- Modelo de base de datos PostgreSQL
- Autenticación JWT
- Validación de archivos
- Dockerfiles para despliegue
- Documentación completa

### 🎯 Features
- Procesamiento de archivos con IA
- Soporte para múltiples formatos (TXT, PDF, CSV, DOCX)
- API RESTful con Swagger UI
- Validación de inputs
- Rate limiting (preparado)
- CORS configurado
